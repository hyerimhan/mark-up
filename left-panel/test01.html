닫기

소방대상물 검색 화면에서 사용 하는 검색 연산자 예시 입니다.

주소 검색 :
경기도 수원시 팔달구 매산로3가 44-2
 - 띄어쓰기로 여러 단어 입력 시 위치에 관계 없이 모든 단어가 포함된 대상물 검색
 - 띄어쓰기는 기본적으로 “AND” 연산 처리(대문자만 인식)
 - 가장 정확도가 높은 순서로 표시

"경기도 수원시 팔달구 매산로3가 44-2"
 - 주소를 쌍따옴표(“)로 감쌀 경우 구, 문장에 언어처리를 적용하여 단어가 순서대로 인접하여 배치된 문서 검색
 - ~도, ~시, ~구 , ~동은 생략 가능
   "경기 수원 팔달 매산로3가 44-2"


한 단어 검색 :
- 검색단어가 포함된 대상물만 검색
EX) 소방청

- "*" 연산자를 사용 할 경우 검색단어와 유사한 단어를 검색
EX) 소방청* , *소방청*


여러단어 검색 :
모두 포함(AND) : 띄어쓰기로 여러 단어 입력 시 위치에 관계 없이 모든 단어가 포함된 문서 검색. “AND”를 반드시 대문자로 입력하여야 함. 띄어쓰기는 기본적으로 “AND” 연산 처리됨.
                     예) 연세대학교 원주

한 단어 이상 포함(OR) : 띄어쓰기로 여러 단어 입력 시 위치에 관계 없이 한 단어 이상이 포함된 문서 검색. “OR”를 반드시 대문자로 입력하여야 함.
                      예) 연세대학교 OR 원주

단어 제외(NOT) : 단어가 포함되지 않은 문서 검색. “NOT”은 반드시 대문자로 검색.
                      예) 연세대학교 AND NOT 원주

문장검색(“”) : 구, 문장을 쌍따옴표(“)로 감쌀 경우 구, 문장에 언어처리를 적용하여 단어가 순서대로 인접하여 배치된 문서 검색.
                     "연세대학교 원주"로 검색  "원주 연세대학교"로는 검색 되지 않음
